# Google Forms 連携 & LINE通知設定ガイド

このガイドでは、Googleフォームの送信データを当アプリケーション（Next.js）に送信し、自動でLINE通知を送るための設定手順を説明します。

---

## 1. Googleフォーム (GAS) の設定

1. **スクリプトエディタを開く**
   - 対象のGoogleフォームを開きます。
   - 右上の「︙（その他）」アイコンをクリックし、「**スクリプト エディタ**」を選択します。

2. **コードの貼り付け**
   - 開いたエディタ（`コード.gs`）の内容を全て消去し、`scripts/gas-script.js` の内容を貼り付けます。
   - **重要:** 14行目付近の `WEBHOOK_URL` を、本番環境のNext.jsアプリのURL（またはngrok等のURL）に変更してください。
     - 例: `const WEBHOOK_URL = 'https://your-app-domain.com/api/webhook/google-forms';`

3. **保存して認証する**
   - ファイルを保存します。
   - 上部の関数選択プルダウンから `testPermissions` を選び、「**実行**」をクリックします。
   - 「承認が必要です」と表示されるので、「権限を確認」→ アカウントを選択 → 「詳細」→ 「（安全ではないページ）に移動」の順に進み、アクセスを許可してください。

4. **トリガー（自動実行）の設定**
   - 左側の時計アイコン（トリガー）をクリックします。
   - 右下の「**トリガーを追加**」ボタンをクリックします。
   - 以下の設定にして「保存」をクリックします：
     - 実行する関数を選択: `onSubmit`
     - 実行するデプロイを選択: `Head`
     - イベントのソースを選択: `フォームから`
     - イベントの種類を選択: `フォーム送信時`
     - エラー通知設定: （お好みで）

以上でGoogleフォーム側の設定は完了です。

---

## 2. LINE Messaging API (LINE公式アカウント) の設定

LINE通知を受け取るためには、LINE公式アカウントを作成し、Messaging APIを有効にする必要があります。

### 2-1. Messaging API を有効にする
1. [LINE Developersコンソール](https://developers.line.biz/ja/)にログインします。
2. 「**プロバイダーを作成**」し、その後「**新規チャネル作成**」から「**Messaging API**」を選択します。
3. 必要な情報（チャネル名、説明、業種など）を入力し、チャネルを作成します。

### 2-2. アクセストークン (`LINE_CHANNEL_ACCESS_TOKEN`) の取得
1. 作成したチャネルの「**Messaging API設定**」タブを開きます。
2. 一番下にある「**チャネルアクセストークン (ロングターム)**」の「**発行**」ボタンをクリックします。
3. 発行された長い文字列をコピーしてください。

### 2-3. ユーザーID/グループID (`LINE_GROUP_ID`) の取得
通知を送る宛先のIDを取得します。

**A. あなた個人のLINEに通知を送る場合 (ユーザーID)**
1. チャネル設定の「**チャネル基本設定**」タブを開きます。
2. 下部にある「**あなたのユーザーID**」をコピーします。

**B. スタッフ用のLINEグループに通知を送る場合 (グループID)**
1. チャネルの「Messaging API設定」タブ内の「グループ・複数人トークへの参加を許可する」を「利用する」にします。
2. 通知を送りたいLINEグループに、作成した公式アカウント（Bot）を招待します。
3. *※グループIDを取得するにはBotから送られてくるWebhookを解析する必要があります。手軽に始める場合は、上記「A」のユーザーアカウントで設定し、後ほど拡張することをお勧めします。*

### 2-4. 環境変数の設定 (`.env`)
アプリケーションにLINEの設定を反映させます。

1. プロジェクトルートにある `.env` ファイルを開きます。
2. 以下の行を追加し、手順2-2と2-3でコピーした値を貼り付けます。

```env
# LINE Messaging API
LINE_CHANNEL_ACCESS_TOKEN="2-2で取得したチャネルアクセストークンをここに貼る"
LINE_GROUP_ID="2-3で取得したユーザーID（またはグループID）をここに貼る"
```

3. アプリケーション（Next.js）を再起動します。

---

## 3. 動作確認

全ての設定が完了したら、実際にGoogleフォームから回答を送信してみてください。
成功すると、設定したLINE宛に「🔔 新規の予診票が届きました！」という通知が届きます。
